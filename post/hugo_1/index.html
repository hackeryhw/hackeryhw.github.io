使用hugo配置博客

hugo简介

Hugo，也就是静态网站生成框架，构建了一套代码生成框架和各种可选的主题。可以帮助我们自动搭建好我们的个人博客网站，不需要关注任何展示的效果，只需要关注博客内容本身即可。是不是很强大。

Hugo主题网站：https://themes.gohugo.io

使用hugo编写博客流程：

  1. 使用markdown编写博客。
  2. 将博客通过hugo编译（自动代码生成）成HTML文件。
  3. 将HTML文件发布到服务器或者托管平台上。

# 创建网站目录结构
>>> hugo new site hackeryhw
>>> cd hackeryhw
>>> ls 
archetypes  content     layouts     resources   themes
config.toml data        public      static

#config.toml 是全局配置文件，在里面可以定义博客地址，构建配置，标题，导航栏等等。
#themes是主题目录，可以下载喜欢的主题，然后配置在config.toml里面。
#content是博客文章的目录，编写的博客内容都存放在此。
#static是博客的静态资源，比如图片
#public是编译后的HTML文件保存处。

# 发布markdown文件
hugo new post/my-first.md

hugo主题：

单独的hugo跑起来样式非常单调，想要丰富的样式还需要学习hugo语法，但是hugo提供了丰富的主题，通过配置主题就可以快速的搭建一个网站。

安装hugo主题：

1
2

git clone https://github.com/olOwOlo/hugo-theme-even themes/even
#这个时候even这个主题就被下载到themes目录下。

配置config：

下载好主题后，主题的exampleSite目录一般会有一个config.toml配置文件。复制到你的根目录下的config.tmol文件，然后根据注释做一些基本的修改。
